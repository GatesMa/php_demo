### 代码规范 

spaphp 采用 PSR-2 代码风格

使用 phpcs 检查
```
./vendor/bin/phpcs -p --standard=PSR2 --ignore=vendor ./src

```

修正
```
./vendor/bin/phpcbf -p --standard=PSR2 --ignore=vendor ./src

```
